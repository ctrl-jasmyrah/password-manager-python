# Password Manager 🔐

A desktop password manager built using Python and Tkinter.

## Features
- Save website credentials
- Generate strong passwords
- User-friendly GUI
- Store login details locally

## Concepts Used
- Python
- Tkinter
- File Handling
- Grid Layout
- Random Module
